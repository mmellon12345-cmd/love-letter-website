# Love Letter — Static Site

Files:

- index.html — main page
- css/style.css — styles (from attachment)
- js/main.js — small script to toggle open/close

Open locally:

1. Double-click `index.html` to open in your browser (Chrome works).

Deploy to Firebase Hosting (quick):

1. Install Firebase CLI: `npm install -g firebase-tools`
2. `firebase login`
3. `firebase init hosting` (select the `love-letter-website` folder)
4. `firebase deploy`

Alternatively host as static files on any static host (GitHub Pages, Google Cloud Storage, etc.).
